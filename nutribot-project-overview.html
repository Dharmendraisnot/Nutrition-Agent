Get your API key at cloud.ibm.com/iam/apikeys · Get Project ID at dataplatform.cloud.ibm.com/wx/home

⚙️ AGENT_INSTRUCTIONS — Customize in app.py

Find AGENT_CONFIG near line 45 in app.py. Changes apply immediately.

AGENT_TONE
"friendly" | "professional" | "motivational" — controls how the bot speaks
AGENT_DIET_FOCUS
"balanced" | "vegetarian" | "vegan" | "keto" | "diabetic"
AGENT_LANGUAGE_HINT
"English" or "Hinglish" — mix Hindi words naturally
AGENT_INDIAN_FOCUS
True / False — prioritise Indian recipes, regional dishes, and spices
AGENT_SAFETY_RULES
List of hard rules always enforced — e.g. no extreme deficits, advise doctor, etc.
MODEL_ID
"ibm/granite-13b-chat-v2" — swap to any available Granite model
GEN_PARAMS.temperature
0.0–1.0 — lower = factual, higher = creative. Default: 0.7
GEN_PARAMS.max_new_tokens
Max tokens per response. Default: 900. Increase for longer plans.

POST/api/clear-chatClear conversation history

🧪 Demo Mode

When IBM_API_KEY or IBM_PROJECT_ID are not set, the app runs in Demo Mode automatically. A yellow banner is shown. Pre-written intelligent responses cover BMI, breakfast ideas, calorie questions, and meal plans. All UI features (BMI calculator, dashboard, family profiles) work normally.
